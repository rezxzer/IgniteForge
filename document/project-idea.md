# IgniteForge - AI-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃў

рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃАрЃарЃБрЃџрЃўрЃљрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў MVP AI-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃљрЃЊ **IgniteForge**. рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃЌрЃљрЃњрЃЮрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ bolt.new-рЃўрЃА рЃЏрЃАрЃњрЃљрЃЋрЃАрЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃЌ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃцрЃўрЃЕрЃарЃћрЃЉрЃўрЃЌ: рЃБрЃцрЃљрЃАрЃЮ рЃБрЃџрЃўрЃЏрЃўрЃбрЃЮ рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ, рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃџрЃљрЃЏрЃљрЃќрЃў UI, рЃЉрЃўрЃџрЃўрЃюрЃњрЃЋрЃљрЃџрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃа рЃЊрЃљ рЃарЃБрЃАрЃБрЃџ рЃћрЃюрЃћрЃЉрЃќрЃћ, рЃЊрЃљ рЃ«рЃћрЃџрЃЏрЃўрЃАрЃљрЃгрЃЋрЃЊрЃЮрЃЏрЃЮрЃЉрЃљ рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, indie hackers-рЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃарЃБрЃАрЃБрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЏрЃАрЃЮрЃцрЃџрЃўрЃЮрЃерЃў.

## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ

- **рЃАрЃљрЃ«рЃћрЃџрЃў**: IgniteForge
- **рЃАрЃџрЃЮрЃњрЃљрЃюрЃў**: "рЃљрЃюрЃЌрЃћ рЃерЃћрЃюрЃў рЃўрЃЊрЃћрЃћрЃЉрЃў. рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃџрЃљрЃЏрЃљрЃќрЃў рЃљрЃърЃћрЃЉрЃў рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ."
- **рЃљрЃдрЃгрЃћрЃарЃљ**: IgniteForge рЃљрЃарЃўрЃА рЃБрЃцрЃљрЃАрЃЮ AI-рЃќрЃћ рЃЊрЃљрЃцрЃБрЃФрЃюрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃљрЃА (рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃљрЃЊ рЃљрЃю рЃарЃБрЃАрЃБрЃџрЃљрЃЊ) рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃАрЃарЃБрЃџрЃцрЃљрЃАрЃЮрЃЋрЃљрЃю, рЃърЃарЃЮрЃЊрЃљрЃЦрЃерЃћрЃю-рЃарЃћрЃЊрЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃљрЃЊ рЃгрЃБрЃЌрЃћрЃЉрЃерЃў. рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃФрЃЋрЃўрЃарЃў рЃбрЃЮрЃЎрЃћрЃюрЃћрЃЉрЃў, рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃБрЃерЃюрЃЮ рЃЊрЃўрЃќрЃљрЃўрЃюрЃў, рЃљрЃарЃљрЃюрЃљрЃўрЃарЃў рЃЎрЃЮрЃЊрЃўрЃарЃћрЃЉрЃўрЃА рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃџрЃЮрЃЉрЃљ (рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃарЃБрЃџрЃў рЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃў рЃЌрЃБ рЃњрЃўрЃюрЃЊрЃљ). рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃљрЃдрЃгрЃћрЃарЃА рЃарЃљ рЃБрЃюрЃЊрЃљ рЃЕрЃљрЃбрЃерЃў, AI рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃБрЃцрЃЌрЃљ рЃЎрЃЮрЃЊрЃА рЃџрЃљрЃЏрЃљрЃќрЃў UI-рЃўрЃЌ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃ«рЃЊрЃћрЃЉрЃљ рЃџрЃљрЃўрЃЋ рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃерЃў, рЃърЃарЃћрЃЋрЃўрЃБ рЃЏрЃДрЃўрЃАрЃўрЃћрЃарЃљрЃЊ рЃЊрЃљ рЃЊрЃћрЃърЃџрЃЮрЃў рЃћрЃарЃЌ рЃЎрЃџрЃўрЃЎрЃерЃў.
- **рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃљрЃБрЃЊрЃўрЃбрЃЮрЃарЃўрЃљ**: рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃћрЃЉрЃў рЃАрЃгрЃарЃљрЃцрЃў рЃърЃарЃЮрЃбрЃЮрЃбрЃўрЃърЃўрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃљрЃарЃљрЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃцрЃљрЃБрЃюрЃЊрЃћрЃарЃћрЃЉрЃў, рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃў рЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃБрЃАрЃБрЃџрЃћрЃюрЃЮрЃЋрЃљрЃюрЃў рЃерЃћрЃЏрЃЦрЃЏрЃюрЃћрЃџрЃћрЃЉрЃў рЃБрЃЎрЃћрЃЌрЃћрЃАрЃў рЃћрЃюрЃЮрЃЉрЃарЃўрЃЋрЃў рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃБрЃюрЃўрЃЎрЃљрЃџрЃБрЃарЃў рЃБрЃърЃўрЃарЃљрЃбрЃћрЃАрЃЮрЃЉрЃћрЃЉрЃў**:
  - рЃЉрЃўрЃџрЃўрЃюрЃњрЃЋрЃљрЃџрЃБрЃарЃў AI: рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЊрЃљ рЃарЃБрЃАрЃБрЃџрЃў рЃърЃарЃЮрЃЏрЃърЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - рЃДрЃЮрЃЋрЃћрЃџрЃЌрЃЋрЃўрЃА рЃџрЃљрЃЏрЃљрЃќрЃў UI: AI рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА Tailwind CSS, Shadcn/UI рЃЊрЃљ Framer Motion-рЃўрЃЌ (responsive, рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў, dark/light рЃарЃћрЃЪрЃўрЃЏрЃћрЃЉрЃў).
  - рЃБрЃцрЃљрЃАрЃЮ рЃЊрЃљ рЃБрЃџрЃўрЃЏрЃўрЃбрЃЮ: рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Groq API-рЃўрЃА рЃБрЃцрЃљрЃАрЃЮ tier-рЃА рЃЊрЃљ Ollama-рЃА рЃџрЃЮрЃЎрЃљрЃџрЃБрЃарЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА (privacy).
  - Smart Explain Mode: рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ AI рЃњрЃљрЃюрЃЏрЃљрЃарЃбрЃљрЃЋрЃА рЃЎрЃЮрЃЊрЃА рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃћрЃюрЃљрЃќрЃћ) рЃњрЃљрЃюрЃљрЃЌрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - рЃЎрЃЮрЃЏрЃБрЃюрЃўрЃбрЃў рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў: рЃЊрЃљрЃўрЃгрЃДрЃћ 10-15 рЃгрЃўрЃюрЃљ-рЃњрЃљрЃЎрЃћрЃЌрЃћрЃЉрЃБрЃџрЃў рЃерЃљрЃЉрЃџрЃЮрЃюрЃўрЃЌ (рЃЏрЃљрЃњ. Todo рЃљрЃърЃў, CRM, рЃўрЃюрЃЋрЃЮрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃбрЃЮрЃарЃў рЃарЃБрЃЉрЃџрЃўрЃА рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ).

## рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў

1. **рЃЕрЃљрЃб рЃўрЃюрЃбрЃћрЃарЃцрЃћрЃўрЃАрЃў рЃърЃарЃЮрЃЏрЃърЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА**: рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃў рЃЋрЃћрЃџрЃў рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃерЃћрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ (рЃЏрЃљрЃњ. "рЃерЃћрЃЦрЃЏрЃћрЃюрЃў Todo рЃљрЃърЃў рЃљрЃБрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃўрЃЌ"). рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃљ рЃўрЃюрЃњрЃџрЃўрЃАрЃБрЃарЃў рЃЊрЃљ рЃарЃБрЃАрЃБрЃџрЃў.
2. **AI рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ**: рЃърЃарЃЮрЃЏрЃърЃбрЃўрЃА рЃњрЃљрЃњрЃќрЃљрЃЋрЃюрЃљ Groq API-рЃќрЃћ (Llama 3.1 рЃЏрЃЮрЃЊрЃћрЃџрЃў). рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃА рЃАрЃарЃБрЃџ React рЃЎрЃЮрЃЊрЃА Tailwind + Shadcn/UI-рЃўрЃЌ рЃџрЃљрЃЏрЃљрЃќрЃў UI-рЃАрЃЌрЃЋрЃўрЃА.
3. **рЃџрЃљрЃўрЃЋ рЃЎрЃЮрЃЊрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў**: Monaco Editor рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў (TypeScript рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ).
4. **рЃњрЃљрЃДрЃЮрЃцрЃўрЃџрЃў рЃърЃарЃћрЃЋрЃўрЃБ**: рЃЏрЃљрЃарЃфрЃ«рЃюрЃўрЃЋ рЃарЃћрЃЊрЃљрЃЦрЃбрЃЮрЃарЃў, рЃЏрЃљрЃарЃ»рЃЋрЃюрЃўрЃЋ iframe-рЃўрЃЌ рЃЏрЃБрЃерЃљ рЃљрЃърЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ (esbuild-рЃўрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў).
5. **рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ**: рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ Supabase-рЃерЃў (рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА). рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ, рЃгрЃљрЃерЃџрЃљ, рЃњрЃљрЃќрЃўрЃљрЃарЃћрЃЉрЃљ URL-рЃўрЃЌ.
6. **рЃћрЃарЃЌрЃў рЃЎрЃџрЃўрЃЎрЃўрЃЌ рЃЊрЃћрЃърЃџрЃЮрЃў**: Vercel/Netlify/Cloudflare Pages-рЃќрЃћ.
7. **рЃљрЃБрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљ**: Supabase Auth (Google/GitHub).
